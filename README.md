# Akshat Portfolio

Personal portfolio website for Akshat Singh Nayal, built with a customized Anime template.

## Live Project Structure

The deployable site lives in the `anime` folder.

## Features

- Responsive single-page portfolio
- Education, projects, and achievements section
- Clickable project and profile links
- Resume page and downloadable PDF resume
- Themed social icons in footer

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Boxicons

## Run Locally

From the repository root:

```bash
cd anime
python3 -m http.server 8080
```

Open:

```text
http://localhost:8080
```

## Deploy on Vercel

- Import this GitHub repository in Vercel.
- Set `Root Directory` to `anime`.
- Leave build and output settings empty (static site).
- Deploy.
